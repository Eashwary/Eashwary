Steps you have to perform:- 

1:- Setup adb service using below command.
sudo apt-get install  android-tools-adb 
verify using 
adb devices
Output:-
List of devices attached
* daemon not running; starting now at tcp:5037
* daemon started successfully
************	device

2:- Setup aapt service using below command.
sudo apt install  aapt
  
3:- You can use your android devices.

4:- Install the attached app inside the device.

5:- Put user name and password inside the app 

6:- you can use coordinates using adb input events
